# Expense Manager Frontend

A beautiful React frontend for the Expense Manager application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Features

- ✅ View all expenses
- ✅ Add new expense
- ✅ Update expense
- ✅ Delete expense
- ✅ Search expenses by name or category

## API Endpoints

The frontend connects to the Django backend at `http://127.0.0.1:8000/api`

Make sure the Django backend is running before starting the frontend.
